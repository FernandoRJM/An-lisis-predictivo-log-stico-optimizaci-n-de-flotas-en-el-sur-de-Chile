# An-lisis-predictivo-log-stico-optimizaci-n-de-flotas-en-el-sur-de-Chile
Este proyecto aplica Ciencia de Datos y Machine Learning para optimizar la logística de transporte de suministros en la industria del salmón (Región de Aysén, Chile). A partir de datos reales de operación de las embarcaciones Sol Pacífico y Costa Pacífico, el sistema predice el riesgo de incidentes en las entregas.
